Разработана многопользовательская автоматизированная система управления СПБПУ организацией на языке программирования С#, включающая базу данных, запросы для ее создания, процедуры и представления для работы с данными, роли и пользователи для каждого типа ролей. Для обеспечения масштабируемости программы и безопасности данных используется трёхуровневая архитектура, в которой слой представления, слой бизнес-логики и слой доступа к данным взаимодействуют друг с другом через интерфейс. Все пароли шифруются с помощью MD5.

Система включает в себя функционал аутентификации и функциональность для каждой роли базы данных, а также функционал логирования действий пользователей в системе для обеспечения возможности отслеживания и анализа изменений. Были проведены тестирование и оптимизация кода, а также произведена настройка сервера базы данных для обеспечения эффективной работы системы.

В процессе разработки системы были учтены требования к безопасности и защите информации, а также применены современные методы и подходы к программированию. В результате была создана функциональная и надежная система управления организацией, готовая к использованию в бизнес-среде.
